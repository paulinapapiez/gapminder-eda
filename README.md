# Gapminder Data Analysis – Exploratory Data Analysis (EDA)

Exploratory data analysis of the Gapminder dataset (1952–2007) focused on identifying relationships between economic development and life expectancy.

## 🔎 Project Overview

This project analyzes global socio-economic data to investigate how GDP per capita, population growth, and regional differences relate to life expectancy.

The analysis includes:
- data exploration and structure validation
- trend analysis over time
- cross-country and cross-continent comparisons
- correlation analysis
- interpretation of socio-economic patterns

The final output is a structured HTML analytical report generated using R Markdown.

---

## 🎯 Analytical Scope

The project addresses:

- Long-term life expectancy trends in selected countries (Japan, United States, India)
- GDP per capita comparison across continents
- Correlation between GDP per capita and life expectancy
- Population growth dynamics by continent
- Country ranking analysis (top and bottom GDP per capita in 2007)
- Temporal comparison (1952 vs 2007)
- Growth analysis of life expectancy by country
- Case study: Cameroon (1952–2007)

---

## 📊 Key Insights

- Strong positive relationship between GDP per capita and life expectancy.
- Significant global improvement in life expectancy between 1952 and 2007.
- Africa remains the continent with the lowest average GDP per capita and life expectancy.
- Population growth is most dynamic in Asia and Africa.
- Economic growth correlates with improved living conditions, but disparities remain.
