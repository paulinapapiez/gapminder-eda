# Gapminder EDA
Exploratory Data Analysis of Gapminder dataset

## Opis
Projekt prezentuje eksploracyjną analizę danych Gapminder w formie interaktywnego HTML.  
Pokazuje trendy w PKB, oczekiwanej długości życia i populacji dla różnych krajów na przestrzeni lat.

## Technologie
- HTML  
- EDA: Gapminder dataset  

## Uruchomienie
Otwórz plik `EDA.html` w przeglądarce, aby zobaczyć interaktywną wizualizację.

## Pliki w repo
- `EDA.html` – interaktywna wizualizacja
